# Filter Rules

This document describes notable filter rules and selectors used across the blocklists.

## Blogroll and RSS widgets

Generic cosmetic selectors used to hide blogroll, RSS feed, and related widget
containers on sites. These are defined in `lists/adblock/Other.txt` and are
covered by the `! Generic Blocks` section.

Examples include:

- `##.plugin-rss`
- `##.blogroll-wrapper`
- `##.plugin-blogroll`
- `##div[id^="blz_rss_"]`
